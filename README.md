# dull-legal

A simple, self-contained static page that serves as the **Privacy Policy & Terms of Use** for the *dull* app.

## Contents

| File | Description |
|------|-------------|
| `index.html` | Single-page legal/privacy document |

## Sections

- **Privacy Policy** — what data is collected, how it is used, and retention/deletion rights.
- **Terms of Use** — license grant, acceptable use, disclaimers, and governing law.
- **Contact** — how users can reach out with questions.

## Usage

Open `index.html` in any browser, or host it on any static-file server (GitHub Pages, Netlify, etc.) and link to it from your app-store listing.
